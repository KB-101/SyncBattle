About PlaySync Arena

PlaySync Arena

Real-time game challenge platform for two players. Challenge friends instantly and play together!

Features

  • Real-time notifications
  • Two-player challenges
  • Easy ID sharing
  • PWA - Install as app
  • Firebase backend
  • Multiple games

How to Use

  1. Set your username
  2. Share your ID with friend
  3. Connect using friend's ID
  4. Send game challenges
  5. Accept challenges to play

Privacy

All data is stored temporarily and automatically cleaned up. No personal data is collected permanently.
